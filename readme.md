Demo application to compare various client side mvc approaches 

Prerequisites
--------------
- node.js >= 0.10
- bower
- grunt

Setup your environment:
-----------------------
-  <code>npm install </code>
-  <code>bower install</code>

Then start the node server:
<code>grunt serve</code>
